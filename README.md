# linux-user-permission-management
A Linux project for managing users , groups , and file permissions.

## Project Description
Linux User Permission Mangement is a Bash Shell Scripting project used to manage users , groups , and file permissions in a linux system.

## Features
- Create User
- Delete User
- View User
- Create Group
- Add User into Group
- Check User Details
- Change File Permission
- Check File Permission
- Maintain Activity Logs

## Technologies Used
- Linus (Ubuntu)
- Bash Shell Scripting

## Commands Used
- adduser
- deluser
- addgroup
- usermod
- chmod
- ls -l
- id

## Project Purpose
The main purpose of this project is to automate Linux user and permission management tasks using shell scripting
