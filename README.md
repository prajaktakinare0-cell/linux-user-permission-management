# Linux-User-Permission-Management
A Linux project for managing users , groups , and file permissions.

## Project Description
Linux User Permission Mangament is a Bash Shell Scripting project used to manage users , groups , and file permissions in a linux system.

## Features
- Create User
- Delete User
- View User
- Create Group
- Add User into Group
- Check User Details
- Change File Permissions
- Check File Permissions
- Maintain Activity Logs

## Technologies Used
- Linux (Ubuntu)
- Bash shell scripting

## Commands Used
- adduser
- deluser
- addgroup
- usermod
- chmod
- ls -l
- id

## Project Purpose
The main purpose of this project is to automate Linux user and permission management tasks using shell scripting
